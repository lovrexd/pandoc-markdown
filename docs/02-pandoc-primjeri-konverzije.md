# Pandoc


## .md > .html
```bash
pandoc dokument.md -o dokument.html
```
Pretvara Markdown datoteku u HTML web stranicu.

---

## .md > .pdf
```bash
pandoc dokument.md -o dokument.pdf
```
Pretvara Markdown u PDF dokument.

---

## .md > .docx
```bash
pandoc dokument.md -o dokument.docx
```
Pretvara Markdown u Word dokument.
